#Проект банковского виджета

## Описание:

Проект банковского виджета - это веб-приложение на Python для управления счетами и банковскими картами.

## Установка:

Клонируйте репозиторий:
```
git clone https://github.com/llifetime/Project01
```

## Использование:

1. Откройте приложение в вашем веб-браузере.
2. Создайте новый счет или банковскую карту.
3. Назначайте фильтр по выполненым операциям, а затем сортируйте по дате.

# Bank Transactions Processor

Программа для обработки банковских транзакций из различных форматов файлов.

## 🚀 Возможности

- Загрузка транзакций из JSON, CSV и XLSX файлов
- Фильтрация по статусу (EXECUTED, CANCELED, PENDING)
- Сортировка по дате (возрастание/убывание)
- Фильтрация по валюте (только RUB)
- Поиск по ключевым словам в описании
- Конвертация валют через внешние API
